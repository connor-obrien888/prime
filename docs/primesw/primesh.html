 12class primesh(prime):
 13    '''
 14        Class to wrap a keras model to be used with the Sheath-trained PRIME-SH architecture (for solar wind prediction see `primesw.prime`). It is recommended to instantiate `primesh` objects in their default configuration:
 15        ```
 16        import primesw as psw
 17        propagator = psw.primesh()
 18        ```
 19        Users will most likely use this class primarily for its `primesh.predict` method. This method is inherited from `primesw.prime` and thus operates the same way.
 20        To generate solar wind predictions from Wind spacecraft data, specify `start` and `stop` times for the desired prediction.
 21        `start` and `stop` are strings with format 'YYYY-MM-DD HH:MM:SS'.
 22        ```
 23        import primesw as psw
 24        propagator = psw.primesh()
 25        propagator.predict(start = '2020-01-01 00:00:00', stop = '2020-01-02 00:00:00')
 26        ```
 27        If using data from an L1 monitor to make predictions, pass the input data using `input` argument.
 28        If `input` is specified, `start` and `stop` should not be (and vice versa).
 29        `input` is also useful for making predicitons from synthetic solar wind data (see inherited method `primesw.prime.build_synth_input`).
 30        For instance, one can predict what the magnetosheath conditions at the magnetopause nose would be if the solar wind flow at L1 was 700km/s:
 31        ```
 32        import primesw as psw
 33        propagator = psw.primesh()
 34        propagator.predict(input = propagator.build_synth_input(vx=-700))
 35        ```
 36        By default, predictions are made at the average middle of Earth's magnetosheath 12.25 Earth Radii upstream on the Geocentric Solar Ecliptic (GSE) x-axis.
 37        One can also specify a position to propagate to besides the default by specifying `pos`:
 38        ```
 39        import primesw as psw
 40        propagator = psw.primesh()
 41        propagator.predict(start = '2020-01-01 00:00:00', stop = '2020-01-02 00:00:00', pos = [11.25, 5, 0])
 42        ```
 43        All positions are in GSE coordinates with units of Earth Radii.
 44        It is not recommended to make predictions outside of the region PRIME-SH was trained on (within 30 Earth radii of the Earth on the dayside).
 45
 46        When instantiating a `primesh` object, one can specify a predefined `model` to be used instead of the automatically-loaded PRIME-SH model. 
 47        In that case, the scaling functions for the input and target datasets (`in_scaler` and `tar_scaler`), the input and target features (`in_keys` and `tar_keys`), and the output features (`out_keys`) must be specified.
 48        The full list of arguments that can be passed to `primesh` is given below.

140    def predict_raw(self, input):
141        """
142        Wrapper function to predict with a keras model. Differs from `prime.predict_raw` by the inclusion of separate location scaling. Not recommended for direct use, see `prime.predict` instead.
143        """
144        loc_scaled = self.loc_scaler.transform(input[:, 11:14]) #Get the target position from input, scale separately
145        input_scaled = self.in_scaler.transform(input)
146        input_scaled[:,11:14] = loc_scaled #Reinsert scaled location
147        input_arr = np.zeros((len(input_scaled) - (self.window - 1), self.window, len(self.in_keys)))  # Reshape input data to be 3D
148        for i in np.arange(len(input_scaled) - (self.window - 1)):
149            input_arr[i, :, :] = input_scaled[i : (i + self.window)]  # Move the 55 unit window through the input data
150        output_unscaled = self.model.predict(input_arr)
151        output = np.zeros((len(output_unscaled), len(self.out_keys)))  # Stage output data to be 2x target dimensions
152        output[:, ::2] = self.tar_scaler.inverse_transform(output_unscaled[:, ::2])  # Mean values
153        output[:, 1::2] = np.abs(self.tar_scaler.inverse_transform(output_unscaled[:, ::2] + output_unscaled[:, 1::2]) - self.tar_scaler.inverse_transform(output_unscaled[:, ::2]))  # Standard deviations
154        return output

156    def predict_grid(
157        self,
158        gridsize,
159        x_extent,
160        framenum,
161        bx,
162        by,
163        bz,
164        vx,
165        vy,
166        vz,
167        ni,
168        vt,
169        rx,
170        ry,
171        rz,
172        y_extent=None,
173        z_extent=None,
174        y = 0,
175        z = 0,
176        loc_mask=False,
177        subtract_ecliptic=False,
178    ):
179        """
180        Generate predictions from PRIME on a grid of points in GSE coordinates.
181
182        Parameters:
183        -----------
184        -    gridsize (float): Spacing of grid points (RE)
185        -    x_extent (list): Range of x values to calculate on (GSE RE).
186        -    framenum (int): Number of frames to calculate (GSE RE).
187        -    bx (float, array-like): IMF Bx value (nT). If array like, must be of length framenum.
188        -    by (float, array-like): IMF By value (nT). If array like, must be of length framenum.
189        -    bz (float, array-like): IMF Bz value (nT). If array like, must be of length framenum.
190        -    vx (float, array-like): Solar wind Vx value (km/s). If array like, must be of length framenum.
191        -    vy (float, array-like): Solar wind Vy value (km/s). If array like, must be of length framenum.
192        -    vz (float, array-like): Solar wind Vz value (km/s). If array like, must be of length framenum.
193        -    ni (float, array-like): Solar wind ion density value (cm^-3). If array like, must be of length framenum.
194        -    vt (float, array-like): Solar wind ion thermal speed value (km/s). If array like, must be of length framenum.
195        -    rx (float, array-like): Wind spacecraft position x value (GSE RE). If array like, must be of length framenum.
196        -    ry (float, array-like): Wind spacecraft position y value (GSE RE). If array like, must be of length framenum.
197        -    rz (float, array-like): Wind spacecraft position z value (GSE RE). If array like, must be of length framenum.
198        -    y_extent (list): Range of y values to calculate on (GSE RE). If None, z_extent must be specified.
199        -    z_extent (list): Range of z values to calculate on (GSE RE). If None, y_extent must be specified.
200        -    y (float, array-like): Y position (GSE RE) that is held constant if y_extent is not specified. Default 0.
201        -    z (float, array-like): Z position (GSE RE) that is held constant if z_extent is not specified. Default 0.
202        -    subtract_ecliptic (bool): Whether or not to subtract the Earth's motion in the ecliptic from Vy. Default False.
203        Returns:
204        --------
205        -    output_grid (ndarray): Array of predicted values on the grid. Shape (framenum, x_extent/gridsize, y_extent/gridsize, 14). Features as in `prime.out_keys`.
206        """
207        x_arr = np.arange(x_extent[0], x_extent[1], gridsize)  # Create a grid to calculate the magnetosheath conditions on
208        y_arr = np.asarray([y]) # This array is overwritten if y_extent is specified
209        z_arr = np.asarray([z]) # This array is overwritten if z_extent is specified
210        if y_extent is None and z_extent is None:
211            raise ValueError("Must specify y_extent or z_extent")
212        if y_extent is not None:
213            y_arr = np.arange(y_extent[0], y_extent[1], gridsize)  # Y positions to calculate the magnetosheath conditions on
214        if z_extent is not None:
215            z_arr = np.arange(z_extent[0], z_extent[1], gridsize)  # Z positions to calculate the magnetosheath conditions on
216        x_grid, y_grid, z_grid = np.meshgrid(x_arr, y_arr, z_arr)  # Create a grid to calculate the magnetosheath conditions on
217        input_seed = np.zeros((len(x_grid.flatten()) * framenum, len(self.in_keys)))  # Initialize array to hold the input data before unfolding it
218        for idx, element in enumerate([bx, by, bz, vx, vy, vz, ni, vt, rx, ry, rz]):  # Loop through the input data and repeat it
219            try:
220                iter(element)  # Check if the element is iterable
221                input_seed[:, idx] = np.repeat(element, len(x_grid.flatten()))  # If it is, repeat it for each grid point
222            except TypeError:  # This error throws if iter(element) fails (i.e. element is not iterable)
223                input_seed[:, idx] = np.repeat(element, framenum * len(x_grid.flatten()))  # If it isn't, repeat it for each grid point *and frame*
224        loc_arr = np.zeros((len(x_grid.flatten()) * framenum, 3))  # Initialize array to hold the location data
225        loc_arr[:, 0] = np.tile(x_grid.flatten(), framenum)
226        loc_arr[:, 1] = np.tile(y_grid.flatten(), framenum)
227        loc_arr[:, 2] = np.tile(z_grid.flatten(), framenum)
228        input_seed_scaled = self.in_scaler.transform(input_seed)  # Scale the input data
229        input_seed_scaled[:, 11:14] = self.loc_scaler.transform(loc_arr)  # Scale the location data
230        input_seed_scaled = np.repeat(input_seed_scaled, self.window, axis=0)  # Repeat the input data 55 times to make static timeseries
231        input_arr = input_seed_scaled.reshape(len(x_grid.flatten()) * framenum, self.window, len(self.in_keys))  # Reshape the input data into the correct shape
232        output_arr = self.model.predict(input_arr)  # Predict the output data
233        output = np.zeros((len(output_arr), len(self.out_keys)))  # Stage output data to be 2x target dimensions
234        output[:, ::2] = self.tar_scaler.inverse_transform(output_arr[:, ::2])  # Mean values
235        output[:, 1::2] = np.abs(self.tar_scaler.inverse_transform(output_arr[:, ::2] + output_arr[:, 1::2]) - self.tar_scaler.inverse_transform(output_arr[:, ::2]))  # Standard deviations
236        output_grid = output.reshape(framenum, len(y_arr), len(x_arr), len(z_arr), len(self.out_keys))  # Reshape the output data into the correct shape
237        output_grid = np.swapaxes(output_grid, 1, 2)  # Move the y axis to the second axis (new order is frame, x, y, z, param)
238        if loc_mask:
239            r_grid = np.swapaxes(np.sqrt(x_grid**2 + y_grid**2 + z_grid**2), 0, 1) # radial distance to origin at all grid points
240            rho_grid = np.swapaxes(np.sqrt(y_grid**2 + z_grid**2), 0, 1) # radial distance to x axis at all grid points
241            theta_grid = np.arctan2(rho_grid, np.swapaxes(x_grid, 0, 1)) # Angular distance to x axis for all grid points
242            output_mask = np.zeros(output_grid.shape, dtype=bool)  # Initialize array to hold the frame mask
243            # Make a mask for all points outside the bow shock or inside the magnetopause
244            for i in np.arange(framenum):
245                if (framenum == 1):  # If there is only one frame turn pdyn and bz into iterables
246                    pdyn_iter = [ni * vx**2 * 1.673e-6]  # Dynamic pressure (nPa)
247                    bz_iter = [bz]
248                else:
249                    pdyn_iter = ni * vx**2 * 1.673e-6
250                    bz_iter = bz
251                bs_array = jelinek_bs(rho_grid, pdyn_iter[i])
252                mp_array = shue_mp(theta_grid, pdyn_iter[i], bz_iter[i])
253                for j in np.arange(len(self.out_keys)):
254                    output_mask[i, :, : , :, j] = (np.swapaxes(x_grid, 0, 1) > bs_array) | (r_grid < mp_array)
255            # Make a masked version of the output grid
256            output_grid = np.ma.masked_array(output_grid, mask=output_mask)
257        if subtract_ecliptic:  # If subtract_ecliptic is true, subtract the Earth's motion in the ecliptic from Vy
258            output_grid[:, :, :, :, 8] -= 29.8
259        return output_grid

261# Analytic surface functions for bow shock and magnetopause
262def jelinek_bs(y, pdyn, r0=15.02, l=1.17, e=6.55):
263    """
264    Bow shock model from Jelinek et al 2012. Assumes GSE Z=0 RE.
265
266    Parameters:
267    -----------
268    -    y (float): GSE Y coordinate (RE)
269    -    pdyn (float): Solar wind dynamic pressure (nPa)
270    -    r0 (float): Bow shock average standoff distance tuning parameter (RE)
271    -    l (float): Lambda tuning parameter
272    -    e (float): Epsilon tuning parameter
273    Returns:
274    --------
275    -    bs_x (float): GSE X position of bow shock (RE)
276    """
277    bs_x = r0 * (pdyn ** (-1 / e)) - (y**2) * (l**2) / (4 * r0 * (pdyn ** (-1 / e)))
278    return bs_x

281def shue_mp(theta, pdyn, bz):
282    """
283    Magnetopause model from Shue et al 1998. Assumes GSE Z=0 RE.
284
285    Parameters:
286    -----------
287    -    theta (float): Polar angle position of desired MP location (radians)
288    -    pdyn (float): Solar wind dynamic pressure (nPa)
289    -    bz (float): IMF Bz (nT)
290    Returns:
291    -    rmp (float): Magnetopause radial distance to Earth (RE)
292    """
293    r0 = (10.22 + 1.29 * np.tanh(0.184 * (bz + 8.14))) * (pdyn ** (-1 / 6.6))
294    a1 = (0.58 - 0.007 * bz) * (1 + 0.024 * np.log(pdyn))
295    rmp = r0 * (2 / (1 + np.cos(theta))) ** a1
296    return rmp

Class to wrap a keras model to be used with the Sheath-trained PRIME-SH architecture (for solar wind prediction see primesw.prime). It is recommended to instantiate primesh objects in their default configuration:

import primesw as psw
propagator = psw.primesh()

Users will most likely use this class primarily for its primesh.predict method. This method is inherited from primesw.prime and thus operates the same way. To generate solar wind predictions from Wind spacecraft data, specify start and stop times for the desired prediction. start and stop are strings with format 'YYYY-MM-DD HH:MM:SS'.

import primesw as psw
propagator = psw.primesh()
propagator.predict(start = '2020-01-01 00:00:00', stop = '2020-01-02 00:00:00')

If using data from an L1 monitor to make predictions, pass the input data using input argument. If input is specified, start and stop should not be (and vice versa). input is also useful for making predicitons from synthetic solar wind data (see inherited method primesw.prime.build_synth_input). For instance, one can predict what the magnetosheath conditions at the magnetopause nose would be if the solar wind flow at L1 was 700km/s:

import primesw as psw
propagator = psw.primesh()
propagator.predict(input = propagator.build_synth_input(vx=-700))

By default, predictions are made at the average middle of Earth's magnetosheath 12.25 Earth Radii upstream on the Geocentric Solar Ecliptic (GSE) x-axis. One can also specify a position to propagate to besides the default by specifying pos:

import primesw as psw
propagator = psw.primesh()
propagator.predict(start = '2020-01-01 00:00:00', stop = '2020-01-02 00:00:00', pos = [11.25, 5, 0])

All positions are in GSE coordinates with units of Earth Radii. It is not recommended to make predictions outside of the region PRIME-SH was trained on (within 30 Earth radii of the Earth on the dayside).
